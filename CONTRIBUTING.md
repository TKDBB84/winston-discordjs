# Contributing

Code coverage is enforced.

For features:

- Please make sure that there are a few tests covering your new feature

For bug fixes:

- Please add a test to reproduce the bug

## Building

```shell script
npm run build
```

## Testing

```shell script
npm run test
```

## Dependencies

- winston-transport
- discord.js

## Development

- ESLint
- Jest (w/ Babel)
- Prettier
- TypeScript
